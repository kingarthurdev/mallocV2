Starter Code:
Please use these interfaces in this starter file in your code. You can upload this to the lab machines (use scp or drag it into VS Code) and then go to the folder your want the starter code to be in and run unzip /path/to/hw6_starter.zip.

Please use this starter code as it will compile your code into a library.
